Welcome to OreSat Linux Node's documentation!
=============================================

A pythonic frame for all OreSat Linux boards.

.. warning::
    This is still a work in progress.

What is OreSat?
---------------

OreSat is a open source CubeSat from Oregon by PSAS (Portland State Aerospace
Society).

.. note::

    If you are looking for more info on OreSat, go checkout https://www.oresat.org/
