# Example App

An example OLAF app

## Run the example app

- Clone the git repo `$ git clone https://github.com/oresat/oresat-olaf`
- `$ cd oresat-olaf`
- Make a virtual CAN bus `$ sudo ./scripts/vcan.sh`
- `$ cd example_app`
- Install dependencies `$ pip install -r requirements.txt`
- Run example `$ python -m example_app`
